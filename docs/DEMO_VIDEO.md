# Demo video (optional artifact)

StackDrop does not ship a hosted demo video. To record a **30–60s** portfolio clip:

1. `npm run dev` (real Tauri build).
2. Place one `.txt`, one text-based `.pdf`, and one `.docx` under **Documents** or **Downloads** with unique words in each body.
3. Launch StackDrop → click **Index library** → wait for completion status.
4. Search by **file name**, then by a **unique word** from file content.
5. Apply **type** and **parse status** filters briefly.
6. Open a result → show **detail** (path, metadata, preview).
7. Click **Re-scan** on one location or run **Index library** again to show refresh.
8. Use OS screen recorder (Win+G on Windows, QuickTime on macOS).

Save as `docs/proof-screenshots/demo-stackdrop.mp4` or attach to your portfolio separately.
